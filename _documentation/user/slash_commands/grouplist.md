---
title: grouplist
syntax: ~
since: 2.4.0
category: Player
permissions: ~
---

The grouplist command shows all the groups that the server has assigned any permissions to. Here is an example of the output:

```
[SERVER->] Group List:
[SERVER->] SOMESERVER.ADMIN
[SERVER->] SOMESERVER.COP
[SERVER->] EVERYONE
[SERVER->] LOCAL.ADMIN
[SERVER->] VERIFIED
```
