---
command: idlist
syntax: ~
since: 2.4.0
category: Admin
permissions:
  - showOthers
---

The idlist command will display the global ID (also known as their bzid) of the players on the server. Unregistered players will not show a global ID. Example of the output:

```
[SERVER->] CasualFriday         :
[SERVER->] SomePlayer           : 179
```
