---
command: kill
syntax: "<slot# | callsign> [reason]"
since: 2.4.0
category: Admin
permissions:
  - kill
---

The kill command will cause the server to destroy a player's tank.

Example without spaces in the callsign:

```
/kill SomePlayer
```

Example with spaces in the callsign:

```
/kill "Six Feet Under"
```
